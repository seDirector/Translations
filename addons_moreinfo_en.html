Most Games: Metamod:Source, Metamod:Source 2, SourceMod, CounterStrikeSharp

seDirector supports installing, upgrading, downgrading, reinstalling, and removing addons. seDirector is smart when changing the addons; it will only install the files necessary to change the versions - it will not remove your config files or plugins.

If an addon is upgraded, downgraded, or reinstalled, a backup will be made prior to taking any action. Look under the addons\\[Backups] directory for the backups created by seDirector.

MAKE SURE to test your server after making addon changes.
To verify Metamod is working, type 'meta version' in console.
To verify SourceMod is working, type 'sm version' in console.
To verify CounterStrikeSharp is working, type 'meta list' in console and look for 'CounterStrikeSharp'.

If you receive an error message after an update has been installed/changed, make sure you're using the latest version. Older versions may not work properly and cause your server to crash or plugins to malfunction.

SourceMod depends on Metamod:Source to work. Make sure to use a version of SourceMod compatible with your version of Metamod:Source. After installing Metamod:Source, the metamod.vdf file is automatically generated specific to your game server.

CounterStrikeSharp depends on Metamod:Source 2 to work. Make sure to use a version of CounterStrikeSharp compatible with your version of Metamod:Source 2. After installing or removing Metamod:Source 2 or CounterStrikeSharp, the gameinfo.gi file is automatically adjusted.

Garry's Mod: ULib, ULX, Wiremod

seDirector supports installing and removing ULib, ULX, and Wiremod. Upgrading/downgrading is not supported. Most of the update-related features are handled by the scripts themselves, so there probably won't be much of a need to upgrade or downgrade anyway.

MAKE SURE to test your server after making addon changes.

If you already have ULib, ULX, or Wiremod installed and you want seDirector to detect it, rename \\addons\\ULib-oldname to \\addons\\ULib, rename \\addons\\ULX-oldname to \\addons\\ULX, rename \\addons\\Wiremod-oldname to \\addons\\Wiremod.

ULX depends on ULib to work. Make sure to use a version of ULX compatible with your version of ULib.
