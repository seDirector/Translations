API Info

If you have a Pro account, you can utilize the seDirector API.

Note
You MUST append the api key to each request: ?apikey=APIKEY_GOES_HERE

Description URI
List servers info http://localhost:10447/api/servers
Start server http://localhost:10447/api/servers/start/<serverNumber>
Stop server http://localhost:10447/api/servers/stop/<serverNumber>
Restart server http://localhost:10447/api/servers/restart/<serverNumber>
Restart (with in-game notification) http://localhost:10447/api/servers/restartnotify/<serverNumber>
Update server http://localhost:10447/api/servers/update/<serverNumber>
Update (with in-game notification) http://localhost:10447/api/servers/updatenotify/<serverNumber>
Start All Servers http://localhost:10447/api/servers/start/all
Stop All Servers http://localhost:10447/api/servers/stop/all

Examples

Start server 2
http://localhost:10447/api/servers/start/2?apikey=abc123

Update server 4
http://localhost:10447/api/servers/update/4?apikey=abc123

Remote Access

You can replace "localhost" with "127.0.0.1" or when accessing remotely, your public IP address.

If you want to access your API remotely over the Internet, to avoid hacking/DDoS attempts, we recommend allowing only specific IP addresses to access port 10447. You can adjust this with Windows Firewall when creating a firewall rule under "Scope" and "Remote IP address".
